# Bakterien Identifikation und Lern App
Diese App dient zur Identifikation verschiedener Bakterien anhand von unterschiedlichen Test und Merkmalen der Bakterien. Ausserdem gibt es einen Lern-Modus für Studentinnen und Studenten.
