# Produkt-Roadmap – Bakterien Identifikation und Lern-App
Diese Roadmap zeigt die geplanten Entwicklungsschritte für unsere App zur Identifikation von Bakterien oder zum Lernen für Studenten und Studentinnen.
## Version 1.0 – Bakterien Identifikation und Lernmodus erstellen
**Bakterien Identifikation:**
- Einteilung von Gram-positiv und Gram-negativen Bakterien
- Weitere Einteilungen mithilfe von Tests zur Identifikation
- Kurze Erklärungen von einzelnen Tests
**Lernen mit verschiedenen Fragetypen**
**Veröffentlichung der App**
## Version 2.0 – Datenspeicherung für den Nutzer während der Session
**Bakterien Identifikation:**
- Speicherung (in Form einer Tabelle), wann welches Bakterium identifiziert wurde

**Lernmodus:**
- Anzeige, wie viel Prozent richtig
- Speicherung (in Form einer Tabelle), wann wie viel Prozent erreicht wurden
